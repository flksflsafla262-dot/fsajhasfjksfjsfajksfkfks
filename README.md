# Discord Anticrash Bot на Python

Готовый бот с prefix-командами без slash-команд:

- `!verify` — создаёт сообщение верификации с картинкой и кнопкой 🔑. При нажатии выдаётся роль `1521582337651904714`.
- `!partners` — создаёт сообщение партнёров в чёрно-сером стиле с картинкой и кнопкой `Перейти к партнёрам`.
- `!partnersadd` — добавляет партнёра в `database/partners.json`. Форматы:
  - через меню: `!partnersadd`
  - быстро: `!partnersadd Название | https://ссылка`
- `!anticrash` — отправляет публичную панель Anticrash с кнопками:
  - `Добавить пользователя`
  - `Выбрать пользователя`
  - `LocalBan`

Владелец по умолчанию: `1389945313225080953`. Ему автоматически доступны все права.

## 2. Токен

Токен в код не вставлен.

Бот читает токен из переменных окружения:

```text
DISCORD_TOKEN
BOT_TOKEN
TOKEN
DISCORD_BOT_TOKEN
```

Если на хостинге нет переменных окружения, можно вставить токен в файл `token.txt`.

## 4. Запуск на bothost.ru

1. Загрузи весь архив/папку на хостинг.
2. В настройках запуска укажи:

```bash
python main.py
```

3. Если панель хостинга просит токен отдельно — вставь его в поле токена/переменных окружения.
4. Если поля нет — вставь токен в `token.txt`.

## 5. Важные настройки Discord Developer Portal

Включи у бота Privileged Gateway Intents:

- Server Members Intent
- Message Content Intent

Без них бот не сможет нормально читать сообщения, выдавать роли и отслеживать участников.

## 6. Права бота на сервере

Боту нужны права:

- Manage Roles
- Manage Messages
- Moderate Members
- Ban Members
- Kick Members
- View Audit Log
- Send Messages

Важно: роль бота должна стоять выше роли верификации и выше ролей, которые он должен снимать/возвращать.

## 8. Как работает Anticrash

### Права пользователей

Файл: `database/users.json`.

Доступные права:

```json
{
  "can_commands": true,
  "can_manage_users": true,
  "can_ban": true,
  "can_mute": true,
  "can_kick": true,
  "can_links": true
}
```

### LocalBan

Если пользователь без права `can_links` отправляет Discord/Telegram/invite-ссылки, бот:

1. Удаляет сообщение.
2. Сохраняет ID и роли пользователя в `database/localban.json`.
3. Снимает все роли, которые может снять, кроме роли `1521582337651904714`.
4. Пишет лог в канал `1521582423702376591`.

Через кнопку `LocalBan` в панели можно выбрать пользователя и вернуть ему сохранённые роли.

### Антиспам

Бот удаляет и выдаёт таймаут на 1 час за:

- слишком длинные сообщения;
- массовые упоминания;
- повторяющиеся символы;
- большое количество спецсимволов;
- частый флуд;
- Discord/Telegram/invite-паттерны.

## 10. Ограничение Discord

Команда с префиксом `!verify` или `!partners` не может сама мгновенно открыть модальное окно по центру экрана. Discord открывает Modal только после interaction-действия: нажатия кнопки, select-меню или slash-команды.

Поэтому реализовано так:

1. Пишешь `!verify` / `!partners`.
2. Бот отправляет служебную кнопку `Открыть настройку`.
3. Нажимаешь кнопку.
4. Открывается центральное окно ввода ссылки.
5. После завершения бот публикует итоговое сообщение.
